About Termon

Termon allows you to monitor your terrarium and get notifications for configurable threshold violations.

To use the Termon API service you need the following things:

  • A Raspberry Pi (Version 2 or 3)
  • An usb Bluetooth dongle (for Pi version 2)
  • One or more Nordic Thingy:52 devices
  • An animal which lives in a terrarium (optional)

You need to setup the Raspberry Pi with Raspbian and install Docker to run our Intermediate Gateway.

If Docker is running on your Raspberry Pi, clone our Intermediate Gateway application from GitHub.

Follow the usage guidelines in the repositories readme file to setup your thingies.
